Effective use of border-radius on some elements can remove the harshness of the square corners on some components. 

## Circle Element

<div class="circle">
</div>

<style>
.circle {
  width: 100px;
  height: 100px;
  border-radius: 50px;
  background-color: #074E68;
  margin: auto;
}
</style>
